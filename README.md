Some good swift extensions.

## Requirements

- iOS 9.0+
- Xcode 8.2+
- Swift 3.0+

## Installation

**.good**swift is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "GoodSwift"
```
